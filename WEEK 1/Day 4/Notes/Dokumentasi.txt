Java Array
- Array digunakan untuk menyimpan beberapa nilai dalam satu variable, daripada mendeklarasikannya satu per satu
- String[] cars untuk contohnya
- setiap lokasi nilai dapat diakses menggunakan indeks, dan pada java indeks dimulai dari angka 0
- untuk mengetahui berapa banyak elemen yang dimiliki sebuah array dapat menggunakan .length
- kita juga membuat array dengan menentukan ukurannya dengan menggunakan new. itu akan membuat array kosong
dengan jumlah ruang tertentu yang bisa kita isikan
- akan tetapi ketika sudah mengetahui nilainya maka bisa langsung tanpa menggunakan "new" dan keduanya sama

Java Method
-Method merupakan sekumpulan code yang diberi nama dan dapat dipanggil kapan saja
public class Main{
    static void myMethod(){
        // code to be executed
    }
}
penjelelasan:
myMethod() merupakan nama dari method
static merupakan method ini merupakan miliki kelas Main dan bukan objek dari kelas Main
void artinya method ini tidak mengembalikan sebuah nilai


// Method with parameter
- Informasi dapat dikirimkan ke method sebagai parameter, dan parameter ini digunakan sebagai variabel
- Parameter ditentukan setelah nama method, di dalam tanda kurung, bisa memiliki banyak parameter, hanya dipisah menggunakan tanda koma
