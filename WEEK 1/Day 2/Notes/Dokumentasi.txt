Java Number
1. Primitive number dibagi menjadi 2 grup:
    a. Integer types : byte, short, int, long
    b. Floating point types : float dan double
2. 
byte: menyimpan angka bulat dari -127  sampai dengan 127, bisa berfungsi untuk menggantikan int apabila nilainya sudah pasti di rentang tsb, berguna untuk menghemat memori
shorts: menyimpan data dari -32768 to 32767
int : menyimpan angka dari -2147483648 to 2147483647
long: menyimpan angka dari -9223372036854775808 to 9223372036854775807, digunakan ketika angka lebih besar dari kapasitas integer
float: digunakan apabila memiliki kebutuhan untuk angka desimal, dan akhiri dengan menggunakan tanda "f" menandakan angkanya float
double: sama case dengan float dan diakhiri dengan huruf "d" untuk menandakan bahwa itu adalah double
3. Keakuratan antara float dan juga double, untuk float memiliki presisi 6-7 digit dibelakang koma sedangkan untuk double hingga 16 digit dibelakang koma
Tipe data boolean
4. boolean data type hanya memiliki 2 nilai yaitu benar dan salah
Tipe data karakter
5. char : digunakan untuk menyimpan karakter tunggal, didalam kutip 1 misalnya 'A' dan 'c'
Tipe data string
String: digunakan untuk menyimpan urutan karakter (text) dan didalam kutip 2

Tipe data non-primitif di Java
6. tipe data non-primitif dinamakan tipe reference dikarenakan tidak menyimpan data secara langsung, tetapi menyimpan referensi (alamat) ke sebuah objek di memori
7. tipe primitif data didefenisikan langsung di bahasa sedangkan tipe non itu dibuat oleh programmer atau library
8. non-primitif digunakan untuk melakukan operasi yang tipe data primitif tidak dapat lakukan
9. primitif dimulai dari lowecase sedangkan untuk tipe data non-primitif dimulai menggunaakn uppercase
10. primitif itu selalu memiliki nilai, sedangkan non-primitif itu bisa menampung null

Java var method
11. Keyword var memungkinkan compiler untuk secara otomatis mendeteksi tipe suatu variable berdasarkan nilai yang kita berikan
12. var akan bekerja ketika kita assign nilai, sebaliknya akan error ketika tidak di assign sebuah nilai


Java Type Casting
Tipe casting di java ada 2:
13. Widening casting : byte -> short -> char -> int -> long -> float -> double (dari yang paling kecil ke yang paling besar)
14. Narrowing casting : double -> float -> long -> int -> char -> short -> byte (dari yang paling besar ke yang paling kecil) dan dimulai dengan tanda ()
