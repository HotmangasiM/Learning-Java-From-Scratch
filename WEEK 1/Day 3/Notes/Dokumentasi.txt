Java Strings
- Variabel yang diapit dengan tanda kutip 2 ("")
- Dikarenakan string adalah sebuah objek maka string memiliki method2 yang bisa melakukan operasi tertentu
cth: 
.length-> untuk mendapatkan value length dari sebuah string
.toUpperCase() -> untuk membuat string menjadi uppercase letters
.toLowerCase() -> untuk membuat string menjadi lowecase letters
.indexOf() -> digunakan untuk mengembalikan index posisi dari string yang didefenisikan (teks)
.charAt() -> method yang digunakan untuk mengakses indeks posisi sebuah karakter
.equals() -> digunakan untuk membandingkan dua strings
.trim() -> digunakan untuk menghilangkan spasi di depan dan belakang string

String Concatenation
- menggunakan tanda + untuk penggabungan string
- bisa melakukan Concatenation juga antara teks dan juga angka
- bisa juga menggunakan method concat-(): untuk concat string
- WARNING!!! tanda + pada string merupakan concat dan tanda + pada bilangan adalah operator

String special karakter
- menggunakan escape karakter \
- \' -> ' -> single quote
- \" -> " -> double quote
- \\ -> \ -> backslash

Other:
- \n -> new line
- \t -> new tab
- \b -> backspace
- \r -> Carriage return
- \f -> Form Feed 

Java Math
- Method- method yang digunakan untuk melakukan operasi matematika

-Math.max(x,y) : mendapatkan nilai tertinggi antara x dan y
-Math.min(x,y) : mendapatkan nilai terendah antara x dan y
-Math.sqrt(x) : mendapatkan nilai akar dari x
-Math.abs(x) : mendapatkan nilai absolute dari x
-Math.pow(x,y) : mendapatkan pangkat yang dihasilkan dari x sebanyak y(multiplication)
Rounding Method
- Math.round(x) : pembulatan ke int terdekat
- Math.ceil(x) : pembulatan ke atas 
- Math.floor(x) : pembulatan ke atas

- Math.random() : membuat bilangan random tetapi menghasilkan double, apabila ingin mendapatkan int harus di casting terlebih dahulu

-  Hanya memiliki 2 nilai yaitu true dan false

Java if ... else
- java conditions dan if statement 
- setiap conditions membutuhkan condisi yaitu false dan juga true (boolean)
- if apabila condition is true, else apabila kondisi salah, else if apabila ada kondisi lain, apabila kondisi pertama false dan switch apabila memiliki banyak alternative blok untuk code 

Java Shorthand (Ternary)
- variable = condition ? expressionTrue : expressionFalse;
- dapat juga menggunakan ternary operation di dalam System.out.println
- nested ternary (ternary bersarang): dapat membuat 2 output, tetapi membuat kode menjadi lebih sulit untuk dibaca

Nested if
- if dalam if yaitu mengecek kondisi jika kondisi lainnya adalah true

Java switch
- Terdapat juga alternatif lain untuk pengecekan pada java dengan menggunakan switch
- berfungsi ketika kita ingin membandingkan nilai satu variabel dengan banyak kemungkinan nilai
switch (expression):
    case 1:
            statement
            break(stop eksekusi);
    default(else):
            statement
- default akan dijalankan apabila tidak ada case yang cocok


Java While Loop
- Loop digunakan untuk mengulang pekerjaan yang sama berkali-kali tanpa menulis kode berulang
- while loop sintaks
while(condition){
    // kode yang diulang
}

Java Do-While Loop
- merupakan varian dari while loop dan memiliki perbedaan yaitu sudah dilakukan eksekusi minimal sekali

- Java For Loop
- ketika mengetahui berapa kali perulangan akan dilakukan maka bisa menggunakan perulangan for

for(statement 1; statement 2; statement 3){
    // kode blok
}
statement 1: dieksekusi sekali, sebelum eksekusi blok kode
statement 2: defines the condition the block code
statement 3: dieksekusi setiap kali setelah blok code dieksekusi

Java Nested Loops
- memungkinkan untuk melakukan loop di dalam loop

Java For Each

Sintaks:
for(type variableName: arrayName){
