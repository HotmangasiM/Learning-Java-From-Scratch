18. terdapat beberapa variable pada bahasa pemrograman java

int => menyimpan integers (whole numbers)
float => menyimpan bilangan decimal
char => menyimpan karakter
boolean => menyimpan 2 state, benar atau salah

cara mendeklarasikan variables;
type variableName = value;
